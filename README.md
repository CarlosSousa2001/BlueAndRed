# 🟥🟦 BlueRed

Boas-vindas ao projeto **BlueRed**!

A proposta da aplicação é oferecer uma experiência moderna e divertida para **criar, responder e compartilhar cartas de perguntas no estilo “Vermelho ou Azul”** (Red Pill or Blue Pill).

## 🚀 Tecnologias

- **Kotlin**
- **Jetpack Compose**
- **Hilt** (injeção de dependência)
- **Ktor Client** (comunicação com backend)
- **Code** (compartilhamento de decks privados)
- **Clean Architecture** organizada por *features*
- **Android Studio**

---

## 📱 Sobre o projeto

O **BlueRed** foi desenvolvido seguindo uma **Clean Architecture por features**, garantindo organização, escalabilidade e facilidade de evolução conforme o projeto cresce.

A aplicação utiliza:

- **UI declarativa com Jetpack Compose**, possibilitando transições fluidas e telas dinâmicas.
- **Ktor Client** integrado ao backend (Fastify + Prisma) para operações como criação de decks, votos e autenticação.
- **Code** para compartilhar decks privados com amigos de forma rápida e segura.
- **Injeção de dependência (Hilt)** para manter o projeto modular, limpo e fácil de testar.

O resultado é um aplicativo envolvente, escalável e alinhado às melhores práticas do desenvolvimento Android moderno, ideal tanto para fins educativos quanto para produção.
